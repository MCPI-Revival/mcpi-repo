# MCPI-Repo Worlds
Repo for Minecraft Pi world sharing. Designed for the MCPIL launcher.

## Installation
Download a world file and extract it. Move the extracted foler to `~/.minecraft-pi/games/com.mojang/minecraftWorlds`.
